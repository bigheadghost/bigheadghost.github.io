Clurichaun序列重排谜题

2020-03-05 12:28:32 CST 约 725 字 预计阅读 2 分钟

文章目录

《世界迷宫10：愚者之财（Labyrinths of the World: Fool’s Gold）》是Domini Games制作、BigFish发行的一款冒险解谜游戏。画风精美，剧情优良，其中有一道谜题 minigame 4 - Clurichaun 很有意思。

图中最内一圈有8个图案，迷宫内部也有8个图案，前后两组图案通过迷宫建立对应关系，内圈每个图案最终要通过变换位置变为它对应的迷宫内的图案。
内圈图案位置则通过操作宝剑来调整。剑上有两个空位置，点击左右箭头移动剑，点击中间箭头，则会将剑槽内的图案 同 其对应的内圈上的两个图案 交换。
通过不断的交换，实现对 内圈图案 的重排。

我们对内圈图案从左到右依次编码为1~8，那么问题就变成：
通过两个辅助位，如何把序列12345678重新排列为63452817？

8个图案加上两个辅助位，一共10个位置，宝剑的位置也有7种，直接求解的话，复杂度较高。
仔细观察起止序列，对数字进行拆解，分成位数较小的，复杂度一下子就降下来了。

1
2
3
4
5

12345 - 678  → 34521 - 678  【6步: {2, 1, 2, 1, 4, 2}】
3452 - 1678  → 3452 - 6817  【没有解，不存在这样的路径】，改成：
345 - 21678  → 345 - 26817  【9步: {2, 1, 2, 1, 3, 4, 1, 2, 3}】
34526 - 817  → 63452 - 817  【7步: {2, 4, 2, 1, 4, 3, 1}】
一共需要22步。

每个数字代表剑的位置（左槽对应的内圈图案位置）。

官方给出的答案是{{6, 1, 6, 2, 4, 2}, {3, 7, 6, 3, 2, 6, 5, 2}}, 一共14步。前6步完成了12345678 → 64573128的变换，后8步完成了64573128 → 63452817。
进一步计算得知：
12345678 → 64573128确实需要6步(具体步骤略有不同)，但64573128 → 63452817最少只要5步即可，一共只要11步。
所以这里就有了一个比官方更好的答案(0代表剑槽)：
{{1, 6, 1, 2, 4, 2}, {5, 2, 3, 7, 5}}
1234567800 => 0034567812 => 0034512867 => 6734512800 => 6004512873 => 6007312845 => 6457312800 => 6457002831 => 6317002845 => 6345002817 => 6345001728 => 6345281700
